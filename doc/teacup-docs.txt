JIRA NEOVIM PLUGIN                            *teacup-docs-jira-neovim-plugin*

A Neovim plugin to interact with Jira, allowing you to fetch and display Jira
tickets directly within your editor.


FEATURES ~

- **Fetch Jira Tickets**Easily retrieve a list of Jira tickets using your API token.
- **Display Tickets**View ticket summaries in a popup window.
- **Customizable**Configure your Jira API token and domain in your Neovim settings.


INSTALLATION ~


USING LAZY.NVIM

Add the following to your `init.lua`

>lua
    {
        "clivern/jira.nvim",
        dependencies = { "nvim-lua/plenary.nvim" },
    }
<


CONFIGURATION ~

Touse this plugin, you need to set your Jira API token and domain in your
Neovim configuration file (`init.lua`):

>lua
    require("jira").setup({
        token = "your_jira_api_token"
        domain = "your_jira_domain.atlassian.net"
    })
<
